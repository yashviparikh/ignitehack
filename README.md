Food Rescue & Donation Matching Platform

A platform connecting restaurants with NGOs to optimize food donation allocations, minimize wastage, and maximize social impact. Built with a focus on real-time updates, intelligent matching, and reliable delivery.

Features

Real-time Donation Updates: Backend implemented with FastAPI + WebSockets for instant updates.

ML-Based Matching Engine: Assigns donations to NGOs based on proximity, food type, NGO reliability, and urgency.

Fallback Routing: Unassigned donations automatically routed to public volunteers to ensure 100% pickup.

Data Handling & Caching: Optimized allocation and caching algorithms for faster, fairer matching.

Tech Stack

Python, FastAPI, WebSockets, SQL / NoSQL

Machine Learning for donation matching

Efficient backend caching and allocation logic

How it Works

Restaurants upload donation details (type, quantity, pickup time).

ML engine calculates best NGO matches.

Unassigned donations are routed to public volunteers.

Real-time updates are pushed to all stakeholders.

System logs and caches donations for performance and reliability.
