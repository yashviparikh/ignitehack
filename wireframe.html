🔄 KEY USER FLOWS

Flow 1: Donor Posts Food

Login → Dashboard → Post Donation → Fill Form → Take Photo → GPS Location → Submit → Confirmation

Flow 2: NGO Accepts Donation

Login → Nearby Donations → Map View → Select Donation → View Details → Accept → Set ETA → Navigate

Flow 3: Live Pickup Process

En Route → Update Status → Arrive → Contact Donor → Load Food → Take Confirmation Photo → Complete

Flow 4: Admin Monitoring

Admin Login → System Overview → Check Analytics → Monitor Active Pickups → User Management
